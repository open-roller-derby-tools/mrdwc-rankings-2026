# MRDWC Rankings 2026

A Directus endpoint extension for MRDWC Rankings 2026.

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Usage

This extension registers the `/mrdwc-rankings-2026` endpoint in Directus. Install it by placing the built output in your Directus project's `extensions/endpoints/` directory.
